# Gerenciador de Tarefas com LiteDB (WinForms)

Aplicativo simples de gerenciamento de tarefas com persistência de dados utilizando o banco local **LiteDB**.

## Funcionalidades

- ✅ Adicionar tarefa
- ✅ Marcar tarefa como concluída (clique duplo)
- ✅ Remover tarefa
- ✅ Salvar e carregar tarefas automaticamente

## Tecnologias utilizadas

- C#
- Windows Forms
- LiteDB

## Como usar

1. Clone o repositório
2. Abra o projeto no Visual Studio
3. Instale o pacote `LiteDB` via NuGet
4. Rode o projeto (`F5`)
